#11 點擊分類按鈕後有樣式變化（active 狀態）
檔案：public/app.js、style.css

JS（點擊時切換 active）：

js
複製
編輯
document.querySelectorAll('.category-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    document.querySelectorAll('.category-btn').forEach(b => b.classList.remove('active'));
    btn.classList.add('active');
  });
});
CSS：

css
複製
編輯
.category-btn {
  background: #FFF5E1;
  color: #8A1C1C;
  border: 1.5px solid #8A1C1C;
}
.category-btn.active {
  background: #8A1C1C;
  color: #fff;
  border-color: #8A1C1C;
}
備註：無論點哪顆，其他要自動移除 .active，只留當前選中者有樣式。
